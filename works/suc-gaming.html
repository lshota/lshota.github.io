SUC GAMING（ゲームサポート・マッチングプラットフォーム）

依頼者と受託者をつなぐスキルマッチングサービス。案件の募集・応募、メッセージ、決済、レビュー、売上管理までを一貫提供。Stripe Connect によるマーケットプレイス型決済の設計・実装を中心に担当。

  • Laravel 8
  • Vue 2
  • Stripe Connect
  • マーケットプレイス決済
  • 2段階認証

概要

依頼者と受託者をつなぐスキルマッチングサービス。案件の募集・応募、メッセージ、決済、レビュー、売上管理までを一貫提供するプラットフォームの開発を担当。 特に Stripe Connect によるマーケットプレイス型決済の設計・実装を中心に、画面実装から管理画面まで担当しました。

技術的課題

  • Stripe Connect のマーケットプレイス決済で、PaymentIntent のオーソリ／キャプチャ、24 時間以内のキャンセル期限、デスティネーション取引とプラットフォーム経由送金の使い分けを整合させる必要
  • 募集型・依頼型で売買の役割が入れ替わる取引フローに対し、注文ステータス遷移・検収／返金条件を一貫制御する設計
  • ログイン・会員情報変更時のメールベース 2 段階認証を既存メール認証フローと共存させる設計

主な担当

  • Stripe PaymentIntent / SetupIntent API と Webhook エンドポイントを設計し、決済・キャプチャ・返金・Connect OAuth 連携を構築
  • 期限切れ決済の自動キャプチャと週次送金のバッチ運用（Artisan スケジューラ）、Connect 残高同期を実装
  • Laravel 8 + Vue 2 による案件一覧・注文詳細・マイページ等の画面実装、Stripe Elements によるカード登録・決済 UI の組み込み
  • 3 階層カテゴリ・タグ・レビュー・フォロー等のマッチング基盤と管理画面（laravel-admin）の構築
  • メール認証・2 段階認証・キューによる非同期メール送信の実装

成果

  • 決済・売上管理・2 段階認証を含む全機能を実装完了
  • 現在クライアント検収中（2026 年 8 月リリース予定）
