歩行解析 AI 管理システム「MePT」（医療 DX）

変形性股関節症等のリハビリ患者の歩行計測データを、理学療法士（PT）がモバイルアプリで記録し、医療機関管理者が Web 管理画面で閲覧・運用する医療 DX システム。

  • Next.js 15
  • Python Lambda
  • DynamoDB
  • Cognito
  • AI 推論連携
  • 医療 DX
MePT

概要

変形性股関節症等のリハビリ患者の歩行計測データを、理学療法士（PT）がモバイルアプリで記録し、医療機関管理者が Web 管理画面で閲覧・運用するシステム。 管理者向け Next.js フロントエンドと、AWS Lambda（Python）による REST API（患者・PT・計測管理、AI 推論連携）の設計・実装を担当しました。

技術的課題

  • 管理者用・PT 用の 2 系統 Cognito ユーザープールによるマルチテナント分離と、API Gateway カスタムオーソライザー（JWT / JWKS 検証）による認可設計
  • DynamoDB の 1MB クエリ制限・1,000 件超の計測データ一覧に対するページネーションと件数集計の効率化
  • S3 上の歩行動画を一時クレデンシャルの署名付き URL で取得し、グラフと動画再生位置を同期表示する UI の実装

主な担当

  • Next.js 15（App Router）による医療機関管理者向け画面（PT・患者 CRUD、計測一覧・推移比較）と BFF API Route の実装
  • Cognito 連携の認証基盤（サインアップ・ログイン・Cookie ベースのミドルウェア保護・トークン再取得）の構築
  • Python Lambda 30 関数超の CRUD API（DynamoDB）と、PT アカウントの Cognito 自動発行の実装
  • 計測イベント登録時に AI 推論 Lambda を非同期起動し、推論結果を DynamoDB へ一括反映するパイプラインの設計
  • Chart.js による関節角度・股関節累積負荷の可視化と、最大 7 件の計測推移比較画面の実装

成果

  • 2025 年 6 月リリース、本番稼働中
  • 医療機関に求められるセキュリティ・権限要件を満たす管理基盤を構築
