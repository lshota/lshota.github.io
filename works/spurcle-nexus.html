Spurcle Community（Nexus）

ビジネスマッチングとイノベーションプログラム向けの集中型 Web プラットフォーム。認証基盤から CMS 設計、バックエンド API、AWS インフラ構築まで一貫担当。

  • Next.js
  • Laravel 8
  • AWS
  • MicroCMS
  • AWS Cognito

概要

特定のビジネスマッチングおよびイノベーションプログラムを支援するための、企業紹介・マッチング機能を備えた Web プラットフォーム。 スピーディーな情報更新と安全な認証を両立することを目的としたプロジェクトです。

役割・担当範囲

SE / 個人担当として、認証基盤から CMS 設計、バックエンド API、フロントエンド連携、AWS インフラ構築までを一貫して担当しました。

  • MicroCMS を採用したヘッドレス CMS の設計と運用フロー構築（非エンジニアでもコンテンツ更新が可能な仕組み）
  • Laravel 8 で API 用バックエンドを開発し、Next.js フロントエンドとの責務分離を明確化
  • AWS Cognito ＋ マツリカフォーム OTP ログイン連携によるセキュアな認証フロー構築
  • AWS EC2 / RDS のインフラ構築、CI/CD パイプライン整備
  • フロントエンド側の API 組み込みおよび画面表示ロジックの実装

工夫した点

  • セキュリティと利便性の両立：Cognito の OTP 連携で、セキュアかつユーザー負担の少ないログイン体験を実現
  • 運用負荷の軽減：MicroCMS の採用により、非エンジニアによるコンテンツ更新が可能となり、開発側の作業を大幅に削減
  • 拡張性のある責務分離：フロントエンド / バックエンド / CMS を明確に分離し、将来の機能拡張・運用保守に耐える構成を構築

結果

  • 非エンジニアによるコンテンツ更新が可能となり、運用負荷を大幅に軽減
  • 認証フローの改善により、セキュリティと利便性を両立したログイン体験を提供
  • 保守性・拡張性の高い構成として、本番環境で安定稼働中
