みんなの競輪

競輪の予想・レース映像を見ながら投票できるオンライン投票サービスの、フロントエンドリニューアル案件。

  • React
  • Styled Components
  • Storybook
  • UI/UX 改善

概要

お得なキャンペーンや特典を提供し、競輪初心者から経験者まで幅広いユーザーが、レース映像や予想情報を確認しながら簡単に投票できる競輪投票サービス。 既存サービスの UI/UX 改善を目的とし、ReactJS（Styled Components）を用いたフロントエンド刷新を担当しました。

役割・担当範囲

SE / 5名チームでの参画。フロントエンドリニューアル全般を担当しました。

  • ReactJS ＋ Styled Components を用いたフロントエンドリニューアル実装
  • コンポーネント設計：デザインとロジックをコンポーネント単位で整理し、UI の一貫性と再利用性を向上
  • 投票画面・レース情報表示画面の UI/UX 改善（投票までの操作ステップを簡略化）
  • Storybook を導入したコンポーネント単位の UI テスト・表示確認体制の構築
  • 既存 API との連携を考慮したデータ取得タイミング・表示制御の設計

工夫した点

  • 初心者にも直感的な UI：競輪初心者でも迷わず操作できる、シンプルな投票導線を設計
  • スタイル管理の刷新：Styled Components で共通コンポーネント化・テーマ管理を行い、保守性を大幅に向上
  • UI 品質の安定化：Storybook 導入で、デザイン崩れや仕様差異を早期に検知できる体制を整備

結果

  • 投票画面の視認性・操作性が向上し、初心者でも利用しやすい UI を実現
  • コンポーネント設計とスタイル管理の改善により、今後の機能追加・UI 改修がしやすい構成を構築
  • フロントエンド刷新によりサービス全体のユーザー体験向上に貢献
