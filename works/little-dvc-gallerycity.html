GalleryCity（リトル・ダヴィンチ）

お子様の絵や工作を写真で残し、家族間限定で共有する温かい Web プラットフォーム。AI 評価・決済機能を備えたフルスタック開発を担当。

  • React (TypeScript)
  • Laravel 12
  • Docker
  • AI 評価
  • Square 決済

概要

お子様の絵画や工作などの作品を写真として残し、家族間限定で共有できる Web プラットフォーム。 作品投稿・共有・コミュニケーション機能に加え、AI による作品評価や決済機能を備えたサービスとして設計。 フロントエンド・バックエンドの両方を担当し、API 連携を含めたフルスタック開発を行いました。

役割・担当範囲

PM 1名 ＋ SE 2名体制で参画。フルスタック開発と API 連携を担当しました。

  • React（TypeScript）でフロントエンド開発：作品一覧、投稿、いいね・コメント、家族専用チャット UI
  • Laravel 12 によるバックエンド開発：作品投稿・家族間共有・コメント・いいね等の API を実装
  • 家族単位でのアクセス制御を実装し、限定共有を実現
  • AI メンターによる作品評価機能の実装（サービスの差別化）
  • Square 決済を導入し、サービス内課金フローを構築
  • Docker を用いた開発環境の統一、GitHub によるソース管理

工夫した点

  • プライバシー重視の限定共有設計：家族単位のアクセス制御で、安心して子どもの成長記録を残せる仕組みを構築
  • インタラクティブ機能の統合：いいね・コメント・チャットなど複数機能を整合性を保ちながら統合的に実装
  • 差別化機能の組み込み：AI 評価・決済機能を安全に統合し、付加価値の高いプラットフォームを実現
  • フロント・バック一貫対応：両者の整合性を一人で担保することで、開発スピードと品質を両立

結果

  • 家族間限定の安全な共有機能により、安心して成長記録を残せるサービスを実現
  • インタラクティブ機能で家族内コミュニケーションを活性化
  • AI 評価・決済機能による継続利用可能なプラットフォームを構築
