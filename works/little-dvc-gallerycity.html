GalleryCity（リトル・ダビンチ）

子どもの絵画・工作などの作品を家族単位で共有・保管し、コンクール応募、ポイントによるいいね、AI による作品評価コメントを提供する Web アプリケーション。

  • Vue 3
  • TypeScript
  • Laravel 9
  • OpenAI GPT-4o
  • Square 決済
  • GCS

概要

子どもの絵画・工作などの作品を家族単位で共有・保管し、コンクール応募、ポイントによるいいね、AI による作品評価コメントを提供する Web アプリケーション。 Vue 3 SPA と Laravel API の前後分離構成で、バックエンド設計（UseCase / DTO パターン）とフロントエンド実装を担当しました。

技術的課題

  • OpenAI API（GPT-4o）との連携における応答遅延時のタイムアウト制御（2 段階リトライ）と、処理中プレースホルダによる重複評価の防止
  • 作品画像のアップロード時に 4 サイズ（サムネイル〜オリジナル）を生成・保存し、DB トランザクションとの整合性を確保
  • 家族・子ども・作品単位の権限判定（コンクール応募状態に応じた閲覧制御を含む）の一貫した適用

主な担当

  • Laravel + UseCase / DTO 構成による REST API 設計・実装（作品 CRUD、コンクール応募、家族管理、チャット、通知）
  • Vue 3 + TypeScript + Pinia による SPA 構築（作品投稿・一覧、決済モーダル、AI 評価 UI、コンクール応募フロー）
  • Intervention Image + Google Cloud Storage による作品画像のマルチサイズ変換・アップロード処理
  • Square API 連携によるポイント購入・応募料・商品購入の決済処理
  • Laravel Socialite（LINE / Google / Facebook / Yahoo）による OAuth 認証と家族招待フローの実装

成果

  • 本番稼働中。プライバシーに配慮した家族限定共有と AI 評価・決済を組み込んだ継続利用型サービスとしてリリース
  • AI 機能（GPT-4o 作品評価）のプロダクト組み込みを設計から実装まで完遂
