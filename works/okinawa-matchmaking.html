婚活マッチング Web システム（沖縄県内向け）

地域の婚活会員向けマッチングサービス。プロフィール登録・お相手検索・マッチング申請・メッセージ交換から、管理者による会員審査・通報対応まで一貫実装。

  • Laravel 10
  • Inertia.js
  • Vue 3
  • PrimeVue
  • マルチガード認証
婚活マッチング

概要

地域の婚活会員向けマッチングサービス。会員はプロフィール登録・お相手検索・マッチング申請・メッセージ交換を行い、管理者は会員審査・マッチング管理・通報対応・コンテンツ配信を行う。 Laravel + Inertia.js + Vue 3 のモノリシック構成で、ユーザー画面と管理画面の双方を実装しました。

技術的課題

  • マッチング申請の状態遷移（申請・承諾・拒否・キャンセル）に加え、同時成立上限・返答期限・期限切れ自動キャンセルを一貫制御する設計
  • 仮登録→メール認証→管理者審査→有料会員化→利用期限管理という多段ステータスの会員ライフサイクル運用
  • 双方向の希望条件照合・ブロック除外・出身地非公開設定など、多数の条件を組み合わせた検索クエリ設計

主な担当

  • Inertia.js + Vue 3 + PrimeVue によるユーザー画面・管理画面の SPA 構成（Vite / Tailwind CSS）
  • ユーザー / 管理者の二重セッション認証（マルチガード）と、メールワンタイムトークンによるログイン認証の実装
  • マッチング申請・成立処理と、返答期限リマインド・期限超過自動キャンセルの日次バッチ（スケジューラ）の実装
  • Eloquent スコープによる会員検索（双方向希望条件・ブロック除外・非公開制御）の設計・実装
  • 管理画面での会員審査・有料ステータス管理、Excel エクスポート機能の実装

成果

  • 本番稼働中。会員向け機能（検索・マッチング・メッセージ）と管理機能（審査・通報・コンテンツ配信）を一通り実装しリリース
