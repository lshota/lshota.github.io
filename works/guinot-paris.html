Guinot 事業向け多階層 Web プラットフォーム

フランス発の高級美容ブランド Guinot のフランチャイズ事業全体を支援する、多階層型 Web プラットフォームをゼロから構築。

  • Laravel 12
  • PHP 8.3
  • Twig
  • AWS
  • 多階層認可

概要

GFJ（Guinot Franchise Japan）、FC（加盟店）、GJ（Guinot Japan）、一般ユーザー向けに提供される、複数の Web サイト群で構成された多階層型プラットフォーム。 施術予約、商品購入、加盟店・店舗管理、上位組織による統合管理など、フランチャイズ事業全体の業務フローを包括的に支援する基幹系 Web システムです。

役割・担当範囲

SE / 2名体制でゼロから開発。要件整理〜実装〜インフラまで一貫対応しました。

  • GFJ / FC / GJ / 一般ユーザー向け各サイトの要件整理および画面仕様策定
  • Laravel 12（PHP 8.3）によるバックエンドをゼロから開発（加盟店・受注・商品・施術予約・決済処理など主要機能）
  • 上位〜下位組織間のデータ同期および業務ワークフローを設計・構築
  • Twig（Laravel テンプレート）を用いた業務画面 UI の実装
  • AWS EC2 / RDS（MySQL）を用いたインフラ構築支援
  • Amazon SES を用いた予約・受注・各種ステータス通知メールの実装

工夫した点

  • 多階層型の権限分離：上位組織から加盟店・一般ユーザーまで、役割ごとに異なる権限・業務フローを安全に分離・統合
  • データ同期の一貫性：組織階層を考慮したデータ管理・反映ルールを実装し、運用ミスや管理工数を削減
  • 業務の一体化：予約・決済・受注・管理を一体化することで、業務の可視化と効率化を実現

結果

  • フランチャイズ事業全体を横断的に管理できる、多階層・業務統合型プラットフォームを構築
  • 組織ごとの権限分離とデータ同期を実現し、運用ミス・管理工数の削減に貢献
  • 要件定義〜実装〜インフラまで一貫対応できる基幹系 Web システム開発の実績を確立
